# 标准参数列表
通过统一使用的参数，可便于替换、复用等操作。
无特殊情况时，建议按照以下规则设置参数。

## 原则
眼睛、嘴巴等部位，正常闭合状态设为0，正常张开状态设为1。

## 提示
如需完全闭合、大幅张开等效果，设置超出0.1单位的范围也可。
可参考“示例模型”进行参数设置。
